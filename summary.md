# Summary — EdgeOne Pages Full-Stack Site Skill

**Name:** EdgeOne Pages Full-Stack Site Skill (`edgeone-pages-fullstack-site`)

**Core value:** Turn a single sentence into a complete, deployable commercial website. The skill guides an AI Agent to scaffold a React + Vite + Tailwind frontend and a modular EdgeOne Pages backend (Edge Functions + Cloud Functions + Middleware + KV), wiring up **Auth, Commerce, Pay, AI Assistant, and Admin Dashboard** as opt-in modules — then hands off to `edgeone-pages-deploy` for one-command go-live.

**Applicable scenarios:**
- Indie hackers shipping an MVP e-commerce or paid-content site in an afternoon.
- SaaS / internal tools needing login + AI + admin without infra glue work.
- Knowledge-payment and membership sites combining Auth + Pay + AI.
- Teams prototyping commercial flows on the EdgeOne edge.

**Shining points:**
- 🚀 One-sentence → full-stack site, runnable locally and deployable globally.
- 🧩 Modular: pick only the modules you need; no dead code.
- ⚡ Edge-native auth (Web Crypto JWT, zero npm deps) for sub-100 ms latency.
- 💳 Stripe-ready with Alipay / WechatPay extension stubs.
- 🤖 LLM-agnostic AI assistant with RAG and intent → action bridging.
- 🔗 Composes cleanly with official `edgeone-pages-dev` / `edgeone-pages-deploy`. MIT.
